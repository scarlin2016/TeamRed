What languages are supported by Visual Studios?

Visual Studio is a complete set of development tools for building ASP.NET Web applications, XML Web Services, desktop applications, and mobile applications. It supports nearly every language with the right plugins (more on that later) but has the support for the following right after install:

  • C
  • C++
  • Visual Basic
  • C#
  • F#
  • XML/XSLT
  • HTML/XHTML
  • JavaScript
  • CSS

Its Best Features

One word: IntelliSense (Wait, maybe 2 words?). IntelliSense is Microsoft's code completion and it can save time and keystrokes. Much like spell check in Office, it also applies a squiggly underlines to code that it doesn't believe is correct.


Newer versions support 'Peek to definition', 'Navigate To' and 'Quick Find' that help you see what's going on with the code you are looking at. This comes in handy especially in huge files with lines and lines of code and it can even display the complete code and where you are in relation to the start and end of it.


Great for debugging the inevitable errors. No matter what you make, whether a Windows program or IOS app, Visual Studio can fix it if it breaks. You can pause the code and check on the value of your variables to make sure everything is moving smoothly. It even lets you move from human readable 'high level' code to 'low level' machine code for particularly pesky issues.


Does have versioning tools and change-highlighting for collabrative work. Visual Studio is available on all major operating systems and can integrate with other popular IDEs like IntelliJ and Eclipse, which allows teams coming from different preferences to stay productive.
